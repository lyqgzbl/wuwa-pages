在线入口：<https://wuwa.lyqgzbl.com/>

## 当前工具

- 成就统计工具：<https://wuwa.lyqgzbl.com/achievement/>
- 任务统计工具：<https://wuwa.lyqgzbl.com/quest/>

## 项目说明

本仓库只包含公开站点入口所需的静态文件，包括首页 HTML、GitHub Pages 配置和自定义域名配置。工具页面由对应的公开仓库生成后部署到本仓库的 Pages 分支目录中。

- `wuwa-quest-tracker`：鸣潮离线任务统计页面生成器
